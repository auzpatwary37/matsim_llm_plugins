# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [0.1.0] - YYYY-MM-DD

### Added
- Initial release for MATSim-libs
- Integration with MATSim for LLM-based agent decision-making
- OpenAI, LM Studio, and Ollama backend support
- Tool framework for AI agents
- RAG support with Qdrant vector database
- Per-agent chat management with conversation history
- Multi-step and parallel tool execution
- Typed tool calling with DTOs
- Three integration modes: replanning, within-day, controller-listener
